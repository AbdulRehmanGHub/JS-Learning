Four Pillars of OOPs

  • Encapsulation. Putting multiple things in 1 place.

  • Abstraction. Hiding the Complexity. Show only important/necessary things and hide unnecessaory info

  • Inheritance. Passing down characteristics from a parennt to child, so new piece of code can be reuse.

  • Polymorphism. Polymorphism means, Poly(many) and Morphism(Behaviours). So, Polymorphism is doing multiple tasks/work with a sinlge function.
